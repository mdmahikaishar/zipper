# Zipper

![Rust](https://img.shields.io/badge/language-Rust-orange.svg)
![License](https://img.shields.io/badge/license-MIT-blue.svg)

Data compression with rust.

## Features

- Zip a file.
- Unzip a file.
- Zip Each entry of a dir.

## Usage

#### Zip a file

```bash
zipper my-file
```

#### Unzip a file

```bash
zipper -u my-file.zip
```

#### Read dir zipping

```bash
zipper -rd
```
